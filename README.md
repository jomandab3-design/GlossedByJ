# Glossed By J
# E-Commerce Website (HTML & CSS)

## Project Overview
This project is a static e-commerce website developed using HTML5 and CSS3 for the CST0400 module.
The website showcases a lip gloss product range and demonstrates structured page design, navigation, layout using CSS Grid, and basic responsive styling.

The website was built without JavaScript, focusing purely on clean semantic HTML structure and CSS styling techniques.

## Website Pages
- Home page (index.html)- Introduction to the brand and navigation menu
- Product page (product.html)- Displays the lip gloss collection using CSS Grid layout
- Contact page (contact.html)- Contact information and enquiry form

## Features Implemented
- Semantic HTML5 structure
- CSS Grid layout for product display
- Navigation bar with styled links
- Product cards with images and descriptions
- Responsive container layout
- Organised folder structure (HTML, CSS, Images)


## Technologies Used
- HTML5
- CSS3
- GitHub Pages
